의사가 배우는 AI · 4편

의료 현장에서 안전하게 프롬프트 쓰는 법

좋은 프롬프트는 AI를 똑똑하게 만드는 문장이 아니라, 작업 범위와 금지선을 분명하게 정하는 지시문에 가깝습니다.

대상: 전공의, 전문의, 교육자 / 난이도: 중급 / 주제: 프롬프트 작성, 윤리, 문서화

이 글의 한 문장 요약

의료 프롬프트의 핵심은 더 많은 답을 끌어내는 것이 아니라, 추정 금지와 검토 범위를 명확히 적어 안전장치를 거는 데 있습니다.

좋은 프롬프트의 기본 구조

  1. 역할을 지정합니다. 예: 의료진의 문서 보조자
  2. 작업 범위를 제한합니다. 예: 요약만 수행, 판단하지 말 것
  3. 출력 형식을 지정합니다. 예: 5문장, 표, 항목 리스트
  4. 금지 조건을 명확히 적습니다. 예: 추정 금지, 모르면 확인 필요 표시

의료에서 특히 필요한 문장

  • 불확실한 내용은 단정하지 말고 "확인 필요"로 표시해 주세요.
  • 원문에 없는 정보는 추가하지 마세요.
  • 환자 친화 언어로 바꾸되 의학적 의미는 유지해 주세요.
  • 가능하면 구조화하되 판단은 하지 마세요.

나쁜 프롬프트의 특징

지시가 넓고 모호하며, 원하는 형식이 없고, 추정을 막는 조건이 없습니다. 이런 경우 AI는 사용자의 기대를 맞추려다 오히려 틀린 내용을 길게 만들어낼 가능성이 높습니다.

실전에서 자주 쓰는 패턴

  • 요약형: 긴 메모를 일정한 형식으로 압축
  • 설명형: 환자 눈높이에 맞게 변환
  • 점검형: 문장 중 위험하거나 확인 필요한 부분 표시
  • 교육형: 발표 목차, 토론 질문, 학습 목표 생성

주의할 점

프롬프트를 잘 쓰는 것보다 중요한 것은 "이 작업을 AI에 맡겨도 되는가"를 먼저 묻는 습관입니다. 금지 영역을 먼저 정한 뒤 문장을 다듬는 것이 맞는 순서입니다.

바로 써볼 프롬프트

당신은 의료진의 문서 보조자입니다. 아래 텍스트를 환자 설명용 문안으로 바꿔 주세요. 조건: - 전문용어는 가능한 쉬운 표현으로 바꿀 것 - 원문에 없는 내용은 추가하지 말 것 - 불확실한 내용은 "의료진 확인 필요"로 표시할 것 - 5문장 이내로 정리할 것 [원문 붙여넣기]

마무리

프롬프트는 기술 문장이 아니라 안전장치입니다. 의료에서는 특히 더 그렇습니다.
